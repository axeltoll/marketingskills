## Skill Clusters

Skills naturally group into clusters that work together:

### Cluster 1: Strategic Foundation
`product-marketing-context` → `value-ladder-design` → `funnel-architecture` → `offer-creation` → `expert-positioning`

*Flow: Capture business context → design value ladder → select funnel type → craft offers → build personal brand*

### Cluster 2: Story & Persuasion
`story-based-selling` → `copywriting` → `marketing-psychology` → `expert-positioning`

*Flow: Build belief architecture → write persuasive copy → apply psychological triggers → deliver through Attractive Character*

### Cluster 3: Funnel Content Generation
`sales-page-architecture` → `copywriting` → `email-sequence` → `webinar-script` → `video-sales-content` → `lead-magnet-strategy`

*Flow: Design page layouts → write copy → create email sequences → script presentations → produce video content → design lead magnets*

### Cluster 4: CRO & Testing
`page-cro` → `signup-flow-cro` → `form-cro` → `popup-cro` → `onboarding-cro` → `paywall-upgrade-cro` → `ab-test-setup`

*Flow: Optimize pages → optimize signups → optimize forms → optimize popups → optimize onboarding → optimize upgrades → test everything*

### Cluster 5: Traffic & Distribution
`paid-ads` → `dream-100-partnerships` → `social-content` → `content-strategy` → `launch-strategy`

*Flow: Run paid campaigns → build partnerships → create social content → plan content strategy → execute launches*

### Cluster 6: SEO & Discovery
`seo-audit` → `programmatic-seo` → `schema-markup` → `competitor-alternatives`

*Flow: Audit SEO → build programmatic pages → add structured data → create comparison content*

### Cluster 7: Measurement & Analytics
`analytics-tracking` → `funnel-analytics` → `ab-test-setup`

*Flow: Set up tracking → measure funnel performance → design experiments*

### Cluster 8: Retention & Growth
`membership-continuity` → `pricing-strategy` → `referral-program` → `free-tool-strategy`

*Flow: Design membership programs → optimize pricing → build referral loops → create free tools for acquisition*

## Recommended Skill Execution Order

When building a complete marketing system from scratch, skills should be activated in this order:

### Phase 1: Foundation (During Onboarding)
1. `product-marketing-context` — Establish business fundamentals
2. `value-ladder-design` — Map the customer ascension journey
3. `offer-creation` — Design the core offer
4. `expert-positioning` — Build the Attractive Character
5. `funnel-architecture` — Select and design the funnel

### Phase 2: Core Content Generation
6. `story-based-selling` — Build the belief map and story bank
7. `copywriting` — Write core marketing copy
8. `sales-page-architecture` — Design page layouts
9. `email-sequence` — Create automated email flows
10. `lead-magnet-strategy` — Design the lead magnet

### Phase 3: Extended Content
11. `webinar-script` — Script presentations (if applicable)
12. `video-sales-content` — Create video scripts
13. `social-content` — Generate social media content
14. `content-strategy` — Plan ongoing content

### Phase 4: Optimization
15. `page-cro` — Optimize page conversions
16. `pricing-strategy` — Optimize pricing
17. `marketing-psychology` — Apply psychological triggers
18. `ab-test-setup` — Design experiments

### Phase 5: Traffic & Growth
19. `paid-ads` — Launch ad campaigns
20. `dream-100-partnerships` — Build partnerships
21. `launch-strategy` — Plan product launches
22. `seo-audit` + `programmatic-seo` — Organic growth

### Phase 6: Scale & Retain
23. `membership-continuity` — Build recurring revenue
24. `paywall-upgrade-cro` — Optimize upsells
25. `referral-program` — Build viral loops
26. `funnel-analytics` — Measure and optimize
